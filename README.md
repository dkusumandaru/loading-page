# loading-page

## Description
The loading page/preloader/loader is a page to display a process when the content page while loaded. The loading page generally visualized by animation to entertaining users.<br>

## Must be include

```text
loading-page/
├── index.html
├── css/
│   ├── bootstrap.min.css
└── js/
    ├── jquery.min.js
    └── bootstrap.min.js
```
